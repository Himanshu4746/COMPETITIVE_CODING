Our new browser has following features
 
You don't need to type 'www.' to open a website anymore.
Though, you still need to type '.com' to open a website.
The browser predicts all the vowels in the name of the website. (Not '.com', though. Again!)
this means you can type the name of a website faster and save some time.

find the ratio of characters you would have typed in our browser, to your normal browser.

First line contains integer T for number of test cases
Second line contains the URL

output
ratio in the form   number of characters typed in our browser/number of characters typed in normal browsers
